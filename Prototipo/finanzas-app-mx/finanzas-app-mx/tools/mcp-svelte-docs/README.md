# MCP Svelte Documentation Tool

## Introduction

The MCP Svelte Documentation Tool is designed to provide developers with easy access to the latest Svelte documentation and tools. This tool enhances the development experience by integrating documentation directly into your workflow, allowing for quick reference and improved code quality.

## Installation

To install the MCP Svelte Documentation Tool, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/finanzas-app-mx.git
   cd finanzas-app-mx/tools/mcp-svelte-docs
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

## Usage

To start using the MCP Svelte Documentation Tool, run the following command:

```bash
npm start
```

This will launch the documentation tool, allowing you to search and access Svelte documentation seamlessly.

## Features

- **Document Search**: Quickly find relevant documentation based on your queries.
- **Resource Access**: Access various sections of the Svelte documentation directly from the tool.
- **User-Friendly Interface**: Designed for ease of use, making it simple to navigate through documentation.
